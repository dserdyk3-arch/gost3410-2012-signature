# ГОСТ Р 34.10-2012. Электронная цифровая подпись

Реализация схемы электронной цифровой подписи (ЭЦП) по стандарту ГОСТ Р 34.10-2012 на языке Python.

**Автор:** Сердюк Д.О.  
**Группа:** МКБ251  
**Университет:** НИУ ВШЭ, МИЭМ  
**Год:** 2026

---

## Описание

Программа реализует полный цикл работы с электронной подписью:

- Генерация ключевой пары (секретный ключ `d`, открытый ключ `Q = dP`)
- Подпись файла
- Проверка подписи

Все операции с точками эллиптической кривой (сложение, удвоение, умножение на число) реализованы вручную.  
Используется криптографически стойкий генератор случайных чисел `secrets`.

**Параметры кривой:** `id-tc26-gost-3410-12-256-paramSetA` (256-битный уровень стойкости).  
**Хэш-функция:** ГОСТ Р 34.11-2012 (Streebog‑256) через библиотеку `gostcrypto`.
